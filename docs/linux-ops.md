File & Directory Operations
Command,Description
ls -l,"Long format listing (view permissions, owners, and timestamps)."
pwd,Print Working Directory.
cd,Navigate between directories.
mkdir / rmdir,Create or remove empty directories.
touch,Create an empty file or update timestamps.
cp / mv,Copy or move/rename files and directories.
rm -r,Recursive removal of files and directories.
ln -s,Create a symbolic link to a source file.
shred,Securely overwrite a file to prevent data recovery.
